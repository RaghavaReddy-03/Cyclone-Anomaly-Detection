# Cyclone Dataset

## Overview
This dataset contains satellite image sequences of tropical cyclones.
Each cyclone folder consists of time-ordered frames representing the
evolution of the cyclone trajectory.

Due to storage constraints, the dataset is not hosted directly in this repository.
